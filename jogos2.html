Jogos para 2 pessoas

7 Wonders Duel


  • Tempo de jogo: 30 min
  • Número de jogadores: 2
  • Nível de complexidade: 5/10

7 Wonders Duel é uma adaptação do clássico 7 Wonders para 2 jogadores. No qual os jogadores competem no desenvolvimento de uma civilização para superar a inimiga em ciência, militarismo ou progresso. Assim os jogadores tentam usar recursos para progredir e impedir o avanço do adversário.


7 Wonders Duel é propriedade intelectual da Galápagos Jogos

Azul


  • Tempo de jogo: 30-45 min
  • Número de jogadores: 2-4(ideal 2)
  • Nível de complexidade: 4/10

Azul é um jogo de construção de azulejos no qual os jogadores competem para criar um grid com linhas de azulejos iguais. Dessa forma os jogadores vão pontuando conforme conseguem criar diferentes tipos de linhas de azulejos. Ao fim, o jogador com a maior pontuação ganha.


Azul é propriedade intelectual da Galápagos Jogos

Jaipur


  • Tempo de jogo: 30 min
  • Número de jogadores: 2
  • Nível de complexidade: 3/10

Jaipur é um jogo de duelo com cartas, no qual os jogadores vão acumulando pontos por meio de fichas. Os jogadores usam suas cartas e as da mesa para poder trocar e vender mercadorias acumulando as fichas que garantirão sua vitória. Jaipur é disputado em 3 rodadas para determinar o vencedor.


Jaipur é propriedade intelectual da Galápagos Jogos
